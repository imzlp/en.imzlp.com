Using Vive Tracker Without HMD

Steps:

  • Install SteamVR, HTC Vive hardware, and drivers
  • Edit SteamVR/resources/settings, change requirehmd to false, save and restart SteamVR
  • Turn on the tracker, restart SteamVR, and you should see the base station and tracker
  • Clone pyopenvr repository and install openvr via pip
  • Use pyopenvr to obtain device information

Offline Update for Oculus Quest 2

  1. Power off the Quest
  2. Press and hold the Power + volume down buttons to enter fastboot mode
  3. Use the volume up and down buttons to select sideload Mode and press the power button to confirm
  4. Connect the Quest to the PC via USB
  5. Use adb commands to check if the connection is successful


6. Use the adb sideload command to flash the new firmware

7. Wait for the flashing to complete; the device will automatically restart
